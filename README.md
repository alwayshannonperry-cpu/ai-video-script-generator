# Viral Video Script Generator

A simple tool that generates short video scripts for TikTok, Instagram Reels, and YouTube Shorts.

## Features

• Script generator  
• Hook, script, caption, hashtags  
• Copy script button  
• Email capture  
• Daily usage limit  

## How It Works

1 Enter email  
2 Enter video topic  
3 Click generate  
4 Copy the script

## Tech Stack

HTML  
JavaScript  
Serverless functions  
Vercel hosting
