# AI Short Video Script Generator

A simple web app that generates short-form video scripts using AI.

Users enter a video topic and the tool generates:
- A viral hook
- A 30 second script
- A caption
- Hashtags

## Tech Used

- OpenAI API
- Vercel hosting
- Simple HTML and JavaScript

## How It Works

1. User enters a topic
2. The site sends the topic to the AI API
3. The AI returns a short-form video script

## Future Features

- Caption generator
- Trending video ideas
